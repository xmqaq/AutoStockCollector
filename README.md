# AutoStockCollector

一站式 A 股 AI 量化分析系统 — 全自动数据采集 · 智能数据治理 · 多模型 AI 决策 · 量化回测 · 盘前竞价雷达 · 可视化工作流

## 功能概览

### 🎯 盘前竞价雷达
- **集合竞价扫描** — 每日 9:25 自动全市场扫描，4 维度打分（跳空/量能/板块共振/乖离率）
- **诱骗识别** — 高开无量/高开回落/低开回升等 6 类诱多诱空检测
- **板块龙头** — 自动识别今日最强板块及其领涨个股
- **强度排序** — TOP 30 强势股列表，联动研报+PA 信号交叉高亮
- **盘中实时追踪** — 腾讯行情 15s 轮询，实时盈亏/最高最低/状态标记
- **自动模拟交易** — 强度≥80+跳空≥3%+无诱骗 → 自动建仓（ATR 动态 SL/TP）
- **14:50 自动平仓** — 尾盘自动卖出，PnL 写回性能统计
- **风控仪表盘** — 持仓/盈亏/敞口/板块分布/距止损距离，手动一键平仓
- **仓位建议** — 基于强度/诱骗/板块的多档位仓位推荐（买入/观察/跳过）
- **历史表现** — 按强度分桶统计胜率/平均收益率，每日记录可回溯

### 📊 价格行为学分析 (PA)
- **图表形态检测** — HH/HL/LH/LL 趋势结构、EMA 均线带、波动率收缩
- **多时间框架融合** — 日线 → 周线 → 月线，HTF 结构成熟度+支撑阻力预警
- **K 线形态** — 吞没/孕线/十字星/三兵等经典形态
- **信号生成** — 综合技术面输出买卖信号强度 + 信号历史追踪
- **回测验证** — 历史 K 线回放信号逻辑，输出胜率/盈亏比/夏普/最大回撤

### 📰 AI 研报分析
- **研报采集** — 东财板块研报批量拉取，自动提取目标价/评级/动量
- **供给侧聚合** — 多板块交叉覆盖，供应链上下游关联
- **基本面筛选** — ROE>5/营收增长>0/PE<200，行业相对 ROE Z 评分
- **AI 合成摘要** — LLM 综合分析评级动量+目标价+市场环境，输出结构化报告
- **与 PA 协动** — 研报推荐 + PA 技术信号冲突检测，辅助决策

### 🤖 AI 实时监控
- **多维度信号** — 技术面(30%) + 舆情(20%) + 资金流(50%) 加权合成
- **价格预测** — 支撑/阻力位、买入区间、卖出目标、止损位
- **信号轨迹** — 历史信号趋势线 + K 线叠加可视化
- **定时轮询** — 每 15 分钟自动扫描持仓+自选股

### 📡 数据中心
- **采集中心** — 全自动采集 A 股 K 线、财务、资金流向、龙虎榜、融资融券、新闻舆情等 13 类数据，智能断点续采、数据校验、缺失补采
- **实时行情** — 腾讯行情接口 / EastMoney 直连双驱动
- **板块流向 / 资金排行** — 板块轮动热力图、个股资金流向排行、主力散户资金对比
- **龙虎榜 / 融资融券** — 每日龙虎榜席位明细、两融余额与趋势
- **新闻舆情** — 多源财经资讯采集，关联个股与板块，支撑事件驱动策略

### 🧠 AI 智能
- **AI 智能中枢** — 多模型统一调度面板，支持 Claude / GPT / 通义千问 / DeepSeek 等大模型，可插拔切换
- **量化选股** — 7 套 AI 选股策略（舆情驱动、资金跟踪、价值选股、板块轮动、趋势融合、反转套利、自选优选），结构化 JSON 输出，全流程可回测
- **融合选股** — 多策略加权融合，AI 动态调整权重，定期优化表现
- **个股深度分析** — 聚合技术面 + 基本面 + 资金面 + 舆情四维度，生成量化评分与 AI 研判报告
- **AI Agent 管理** — 多 Agent 协作对话、Multi-Agent 分析面板
- **AI Key 管理** — 多厂商 API Key 统一管理，优先级调度、频次限额、容错轮转

### 🔄 量化交易
- **模拟交易** — 内置交易引擎，按 A 股规则计算佣金/印花税，校验交易时段，支持 SL/TP 自动退出
- **仓位管理** — 持仓盈亏追踪、收益曲线图表、实时价格联动、逐日盯市口径
- **工作流管理** — 可视化拖拽编排选股/监控/分析工作流，24 种节点类型，支持定时调度和自定义节点
- **组合快照** — 每日自动记录净值/持仓/现金，生成收益曲线

### 👥 用户管理
- **JWT 认证** — 登录/注册页面，Token 鉴权
- **管理员面板** — 用户列表/禁用/权限管理
- **排行榜** — 模拟交易收益率排行

## 快速开始

### 环境要求

- Python 3.10+
- Node.js 22+ & pnpm
- MongoDB 5.0+

### 本地开发

```bash
# 1. 克隆项目
git clone https://github.com/xmqaq/AutoStockCollector.git
cd AutoStockCollector

# 2. 后端
cd AutoStockCollector-manage
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env   # 编辑 .env 填入 MongoDB 连接串和 AI API Key
python main.py          # 启动后端 http://localhost:5555

# 3. 前端（新终端）
cd AutoStockCollector-web
pnpm install
pnpm dev                # 启动前端 http://localhost:5173
```

### Docker 部署

```bash
# 配置环境变量
cp AutoStockCollector-manage/.env.example AutoStockCollector-manage/.env
# 编辑 .env 填入 MONGODB_URI、AI API Key 等

# 一键启动
docker compose up -d --build

# 访问
# 前端：http://localhost:8888
# 后端 API 通过 Nginx 自动代理到 /api/*
```

### 环境变量

| 变量 | 说明 | 示例 |
|------|------|------|
| `MONGODB_URI` | MongoDB 连接串 | `mongodb://user:pass@host:27017/stock_collector?authSource=admin` |
| `MONGODB_DATABASE` | 数据库名 | `stock_collector` |
| `FLASK_DEBUG` | 调试模式 | `false` |
| AI 相关 Key | 通过 Web 界面「AI Key 管理」页面配置即可 | — |

## 核心设计

### 数据采集策略

- **双数据源驱动**：EastMoney 直连为主（高并发低延迟），腾讯行情为辅（盘中实时），AKShare 备选降级
- **三层缓存**：进程 LRU → MongoDB → API → AKShare 降级，逐级退化
- **智能续采**：基于数据校验结果精准定位缺失区间，仅补采缺失数据，不做全量重采
- **防限流风控**：场景化控频 + 指数退避 + 熔断休眠 + 冷热分流，纯代码实现，无需代理 IP

### 盘前竞价雷达设计

- **扫描流程**：全市场快照(5200+) → 行业归属 → 板块平均跳空 → 金额预排序 → 逐只打分 → TOP 30
- **4 维度权重**：跳空(40%) + 量能排名(30%) + 板块共振(20%) + 乖离率(10%)
- **诱骗检测**：高开无量/高开回落>3%/极端跳空等 6 类规则，纯 9:25 数据无需盘中价格
- **风控体系**：ST/退市/次新股过滤 → 单日最大 5 只 → 总敞口≤现金 80% → 单板块≤30%
- **平仓闭环**：14:50 自动卖出 → PnL 记录 → 性能统计分桶 → 历史胜率可视化

### AI 多模型架构

- **统一适配器**：屏蔽厂商接口差异，新增模型只需实现适配层
- **差异化调度**：核心分析调用高端模型，批量筛选调用高性价比模型
- **容错轮转**：单模型异常自动轮转备用模型，全模型不可用时降级为规则引擎
- **结果缓存**：同日同标的同策略结果去重复用，控制成本

### 数据校验体系

- 时序连续性校验（基于交易日历）
- 核心字段完整性校验
- 数据合法性校验（极值/负数/重复过滤）
- 完整度评分 + 闭环复核
